# **Introduce**
- **‘기술 발표’를 이용해 개인의 기술 역량 향상을 목표하는 모임입니다.**
- 개인의 학습/정리는 간단한 정리에 끝나는 경우가 있습니다. 이럴 때 남들 앞에서 발표를 해야 한다면 조금이라도 더 깊게 알아보게되는 것 같습니다. **이 점을 활용하여 최대한 깊은 고민을 하며 딥다이브하고 성장하는 것을 목표합니다.**
- 본인의 관심사뿐 아니라 다양한 관심사들을 접하며 시야를 넓히는 것도 목표합니다.
